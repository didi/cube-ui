## Loading

加载，提供了可自定义大小的加载动画。

### 示例

- 图标大小

  默认大小为`24px`，可通过`size`属性配置

  ```html
  <cube-loading></cube-loading>
  <cube-loading :size="28"></cube-loading>
  <cube-loading :size="40"></cube-loading>
  ```

### Props 配置

| 参数 | 说明 | 类型 | 默认值 |
| - | - | - | - |
| size | 加载图标的大小，单位px | Number | 24 |
